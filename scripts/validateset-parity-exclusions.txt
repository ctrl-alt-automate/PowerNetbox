# Known-OK ValidateSet discrepancies.

# Format: <relative-path-under-Functions/>::<ParameterName>
# Lines starting with '#' and blank lines are ignored.
#
# Every entry here should have a reason — these are acknowledged exceptions
# where PowerNetbox intentionally differs from NetBox's ChoiceSets, not
# bugs waiting to be fixed.
#
# When adding a new entry, include a comment explaining WHY it is a
# legitimate deviation. When NetBox adds a value that was previously an
# exclusion, remove the entry and let the finding surface.

# ---------------------------------------------------------------------------

# -Method is an HTTP verb list, not a NetBox enum. The script picks
# WebhookHttpMethodChoices by value overlap, but these are used for the

Helpers/InvokeNetboxRequest.ps1::Method
Helpers/Send-NBBulkRequest.ps1::Method

# -Face on Export-NBRackElevation includes 'Both' as a PowerNetbox-only

# -Button_Class on CustomLink is Bootstrap button variants (primary,

Extras/CustomLinks/New-NBCustomLink.ps1::Button_Class
Extras/CustomLinks/Set-NBCustomLink.ps1::Button_Class
